HTML

TEXT

Headings:

<h1 - h6> ... </h1 - /h6>

Paragraphs:

<p> ... </p>

Bold & Italic:

<b> example </b>, <i> example </i>

Superscript & Subscript:

<sup> CO2 </sup>, <sub> H2O </sub>

Line Breaks & Horizontal Rules:

<br />, <hr />

Strong & Emphasis:

<strong> I'm STRONG </strong>, <em> I'm EM </em>

Quotations:

<blockquote> ... </blockquote>, <q> Q </q>

Abbreviations & Acronyms:

<abbr> ... </abbr>, <acronym> </acronym>

Citations & Definitions:

<cite> this is cite </cite>, <dfn> this is dfn </dfn>

LISTS

Ordered Lists:

<ol> <li>123</li> </ol>

Unordered Lists:

<ul> <li>-</li> </ul>

Definition Lists:

<dl> <dt> <dd> TEXT </dd> </dt> </dl>

Nested Lists:

<ul> <li> <ul> <li> </li> </ul> </li> </ul>

LINKS

Linking to other sites:

<a href="https//www.example.com"> Click Me </a>

Linking to other pages on the same site:

<a href="index.html" target="_blank, _self"> Click Me </a>

Enail Links:

<a href="example@gmail.com"> Send Email </a>

IMAGES

Adding Images:

<img src="path of picture" alt="Definition of picture" title="Show Definition when hover" />

Height & Width of Images:

<img width="width of the image in pixels." height="height of the image in pixels." />

TABLES

Basic table structure:

<table> Table Element <tr> Table row <th> Table Head </th> <td> Table Data </td> </tr> </table>

Long Tables:

<thead> The headings of the table </thead>

<tbody> The body s </tbody>

<tfoot> The footer </tfoot>

FORMS

Form structure:

<form action="URL for the page on the server" method="get or post"> Form Body </form>

Text Input:

<input type="type attribute" name="identifies the form control what is it" maxlength="limit the number of characters"> input Body </input>

Text Area:

<textarea cols="column" rows="row"> Textarea Body </textarea>

Password Input:

<input type="password" name="" size, maxlength > input Body </input>

Radio Button:

<input type="radio" value="indicates the value that is sent to the server" name="" checked="checked" > input Body </input>

Checkbox:

<input type="checkbox" name="" value="" checked="" > input Body </input>

Drop Down List Box:

<select name="" > <option value="" selected> Menu List </option> </select>

File Input Box:

<input type="file" name="" > input file </input>

SubmitButton:

<input type="submit" name="" value="control the text that appears on a button" > input file </input>
